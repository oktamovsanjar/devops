# 🐍 Track: Python for DevOps

> Har kuni asosiy mission yonida **10-15 daqiqalik** Python mini-mashq.
> Maqsad: DevOps'da kerak bo'ladigan Python — avtomatlashtirish, API, parsing, CLI.
> (To'liq dasturchi emas — **avtomatlashtiruvchi muhandis** bo'lasan.)

## Nega Python?
Bash kichik ishlar uchun zo'r, lekin murakkab logika, JSON/API, ma'lumot
ishlash kerak bo'lganda Python — DevOps'ning #2 tili (1-si Bash/YAML).

## Yo'l xaritasi (mission'larga parallel)
| Bosqich | Mavzu |
|---------|-------|
| 1 | Asoslar: o'zgaruvchi, type, f-string, input/print |
| 2 | Ma'lumot tuzilmalari: list, dict, set, tuple |
| 3 | Shart va sikl: if, for, while, comprehension |
| 4 | Funksiya, modul, `import`, `__main__` |
| 5 | Fayl bilan ishlash: read/write, `with`, path |
| 6 | JSON va `subprocess` (buyruq chaqirish) |
| 7 | `argparse` — o'z CLI vositangni yozish |
| 8 | HTTP/API: `urllib`/`requests`, status code |
| 9 | Xato boshqaruvi: `try/except`, logging |
