# 💻 Track: Shell Scripting Mastery

> Har kuni **10-15 daqiqa** qo'shimcha shell mashqi (asosiy mission ustiga).
> Maqsad: bir qatorlik (one-liner) "jodugar" buyruqlardan to'liq avtomatlashtirish
> skriptlarigacha. DevOps = avtomatlashtirish, avtomatlashtirish = shell.

## Nega alohida track?
Shell asosiy rejada ham bor, lekin **mahorat takrordan keladi**. Bu track —
har kuni bitta yangi "qurol" yoki one-liner bilan refleksingni o'tkirlaydi.

## Yo'l xaritasi
| Bosqich | Mavzu |
|---------|-------|
| 1 | Pipe & redirect: `\|`, `>`, `>>`, `2>`, `&>` |
| 2 | Filtrlar: `grep`, `cut`, `sort`, `uniq`, `wc`, `head`, `tail` |
| 3 | `sed` — matn almashtirish va tahrirlash |
| 4 | `awk` — ustun va hisob-kitob |
| 5 | O'zgaruvchi, quoting, command substitution `$(...)` |
| 6 | Shart: `if`, `test [ ]`, `[[ ]]`, exit code |
| 7 | Sikllar: `for`, `while`, `read` bilan fayl o'qish |
| 8 | Funksiya, argument `$1 $@`, `getopts` |
| 9 | Xato boshqaruvi: `set -euo pipefail`, `trap` |
| 10 | Real skript: backup / monitor / deploy avtomatlashtirish |

## Kunlik format (mission ichidagi 💻 Shell Drill)
- 1 ta **yangi one-liner** (tushuntirish bilan)
- 1 ta **kichik vazifa** (o'zing yoz)
- Yechimlaringni `tracks/shell/dayXX.sh` ga saqla

> 🏆 Maqsad: kun oxiriga real foydali skript yozadigan darajaga yetish.
